# Compiler

## The Books

This *mini* project is a Rust translation of the ***Thorsten Ball's*** *"Writing An Interpreter In Go"*, covering the `Lexer` and `Parser` and more from the educational *Monkey Language* programming language.

After implementing the lexer and parser, we'll jump directly to his next book *"Writing a compiler in Go"* instead of going through the `Evaluation` chapter, only revisiting it if needed."
